# Diabetes Prediction Using Machine Learning

## Project Overview
This project predicts whether a person is diabetic or not using machine learning algorithms trained on medical data.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Model Training
- Prediction System
- Performance Evaluation

## Dataset
Pima Indians Diabetes Dataset

## Results
The trained model can predict diabetes based on health-related input features.
